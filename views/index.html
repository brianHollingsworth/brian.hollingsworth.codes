Profile.

If it wasn't apparent by the URL or the Bootstrap jumbotron, I'm Brian Hollingsworth. This site is a little side project of mine to add some interactive flair to my resume.
I decided to make this site for a number of reasons, but it mostly boiled down to this: I got bored of LinkedIn. I knew nothing would make me more homogeneous than flaunting my profile on the same platform as every other developer out there. Thus, out of that irritancy, brian.hollingsworth.codes was born!

About brian.hollingsworth.codes

Initially, this MEAN stack application was built on top of Red Hat's OpenShift container platform. However, after the first couple months, the cluster on which my app resided became unreliable and would regularly experience extended outages. I spent a few weeks testing out the big name cloud platforms but ended up moving my app over to DigitalOcean, and it's been smooth sailing ever since!
All of my code is open source, so feel free to look under the hood of my "resume" and fork it if you want!
By the way, brian.hollingsworth.codes utilizes MongoDB to track the number of views it receives.
